This example allows to read mobile beacon's location and raw IMU data via UART from mobile beacon or modem.

To get raw IMU data from mobile beacon you need to enable 'Raw inertial sensors data' option 
in 'Interfaces' section for mobile beacon in dashboard.

To get raw IMU data from modem you need to enable 'IMU via modem' options
in 'Interfaces' section for mobile beacon in dashboard.
