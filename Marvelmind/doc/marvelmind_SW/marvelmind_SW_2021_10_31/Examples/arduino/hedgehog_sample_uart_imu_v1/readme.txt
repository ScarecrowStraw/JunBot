This example allows to read mobile beacon's location and IMU fusion data via UART from mobile beacon or modem.

To get IMU fusion data from mobile beacon you need to enable 'Processed IMU data' option 
in 'Interfaces' section for mobile beacon in dashboard.

To get raw IMU data from modem you need to enable 'IMU via modem' options
in 'Interfaces' section for mobile beacon in dashboard.
